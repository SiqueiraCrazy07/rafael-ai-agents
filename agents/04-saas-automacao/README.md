# 04 - SaaS Automação

Frente responsável pela construção de uma plataforma própria de automação inspirada em Make e n8n.

## Agentes planejados

- Arquiteto SaaS
- Workflow Builder
- Conectores
- Scheduler
- Executor
- Billing

## Objetivo

Criar uma plataforma escalável de automação com agentes, workflows, integrações e execução distribuída.
